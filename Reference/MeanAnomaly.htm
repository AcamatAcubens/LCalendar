MeanAnomaly

Die MeanAnomaly-Funktion liefert die mittlere Anomalie der mittleren Planetenbahn zur julianischen Tageszahl.

Definition

public static double MeanAnomaly(
   [double jd]   // Julianische Tageszahl.
)

Argumente

jd Gibt die julianische Tageszahl des Berechnungszeitpunktes an. Wird das Argument nicht angegeben, wird zur Berechnung das aktuelle Systemdatum verwendet.

R�ckgabewerte

Der Rückgabewert der MeanAnomaly-Funktion enthält das Bahnelement in Gradmaß und bezogen auf das Äquinoktium des Datums. Als Zeitmaß findet TDB Anwendung.

Bemerkungen

Die Berechnung erfolgt nach den Algorithmen von Meeus Kapitel 31 – Elements of the Planetary Orbits.
