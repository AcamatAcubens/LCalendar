Eccentricity

Die Eccentricity-Funktion liefert die Exzentrizitält der mittleren Planetenbahn zur julianischen Tageszahl.

Definition

public static double Eccentricity(
   [double jd]   // Julianische Tageszahl
)

Argumente

jd

Gibt die julianische Tageszahl des Berechnungszeitpunktes an. Wird das Argument nicht angegeben, wird zur Berechnung das aktuelle Systemdatum verwendet.

Rückgabewerte

Der Rückgabewert der Eccentricity-Funktion enthült das Bahnelement bezogen auf das Äquinoktium des Datums. Als Zeitmaß findet TDB Anwendung.

Bemerkungen

Die Berechnung erfolgt nach den Algorithmen von Meeus Kapitel 31 – Elements of the Planetary Orbits.
